# Constructing short-term projections from annual EU SPF forecasts

We construct real-time compatible quarterly forecasts for growth in the EU, using annual consensus forecasts from the ECB Survey of Professional Forecasts (SPF).

## Contents of this repository
The folders contain the following:

* `R`: Contains the fundamental R code called in `scripts` (mainly Kalman filter code as well as helper / utils)
* `data`:
  * `raw`: raw data files as downloaded from data portals (see `documentation` for corresponding sources and links)
	* `helpers`: auxiliary data files needed for matching / mapping
  * `processed`: processed data files as generated by `scripts/01_process_raw`
* `documentation`: text file listing sources / links for data, as well as any additional pdf documentation provided by data portals
* `output`: filtered quarterly spf forecasts
* `plot_results`: generated plots
* `plot_scripts`: scripts to generate plots
* `scripts`: action scripts producing data and analyses
